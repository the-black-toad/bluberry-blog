I'm building an autonomous blueberry picking robot from scratch — cameras, computer vision, machine learning, and eventually a robotic arm that can harvest ripe berries without human input.

This blog documents the whole process: the design decisions, the dead ends, the things that worked on the first try and the things that took a week to debug. The goal is to capture the *why* behind each choice, not just the what.

**The stack so far:** three USB webcams, stereo depth estimation, YOLOv8 for berry detection, Open3D for 3D localization, and a SO-101 robot arm for picking.

Find the code on [GitHub](https://github.com/the-black-toad).
